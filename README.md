# High-Speed_USB_Magnetic_Switch_Keyboard

#### 介绍
Introduces demos of AT32 MCU-based High-Speed USB wired magnetic switch keyboard, featuring an 8KHz polling rate, 16KHz scanning rate and keyboard sensitivity fine-tuning.
